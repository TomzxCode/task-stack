# Keyboard Shortcuts

## Global

| Action | Shortcut |
|---|---|
| Toggle window visibility | **Ctrl+Shift+T** (configurable) |

## Entry Field (top of window)

| Action | Shortcut |
|---|---|
| Add task to top (becomes current) | **Enter** |
| Add task as next (after current) | **Shift+Enter** |
| Add task to top | **Home** |
| Add task to bottom | **End** |
| Cancel edit | **Escape** |

## Task List

| Action | Shortcut |
|---|---|
| Select row by index | **0**-**9** |
| Move selection up / down | **Up** / **Down** |
| Reorder selected row | **Left** / **Right** |
| Promote selected to top | **Home** |
| Send selected to bottom | **End** |
| Edit selected task text | **Return** |
| Delete selected task | **Backspace** / **Delete** |
| Show keyboard shortcuts | **?** |
| Hide window | **Escape** |
| Start typing (redirects to entry) | Any printable character |

## Description Panel

When a task is selected, a description panel appears below the task list.

- Click the panel to type a description for the selected task.
- The description is saved automatically when you click elsewhere or press
  **Escape**.
- Tasks with a description show a note indicator in the task row.
